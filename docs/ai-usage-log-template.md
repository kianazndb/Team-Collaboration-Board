# AI Usage Log Template

Document how you collaborated with AI tools throughout the assignment. Copy this file to something like `docs/ai-usage-log.md` in your working branch and fill it in as you go.

## How To Use This Log
- Add a new entry each time you consult an AI assistant (Cursor, ChatGPT, Copilot, Claude, etc.).
- Summaries should explain what you asked, what you received, and how you applied or rejected the suggestion.
- Feel free to paste snippets or links to relevant commits.
